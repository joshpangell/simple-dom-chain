Simple DOM chain demo

This is a demo for the simple dom chain analyzer. A stroke is applied to the hovered area. Clicking on an element will reveal all of it's parents, including class name and ID. After initialization, links are inactive.

html, body

$(document).ready(function() {
	$("*").DomInspector({
		callback : function(results) {
			// Returns an Array of each element
			$(".Results").html(results.join(", "));
		}
	});
});
